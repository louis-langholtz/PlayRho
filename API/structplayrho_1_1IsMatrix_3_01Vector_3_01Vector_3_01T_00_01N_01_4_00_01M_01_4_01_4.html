Detailed Description

template<typename T, std::size_t M, std::size_t N>
struct playrho::IsMatrix< Vector< Vector< T, N >, M > >

Trait class specialization for checking if type is a matrix type.

Trait class for determining whether the given type is a matrix, that is, a vector of vectors.

Note
This implements the specialized case where the given type is indeed a matrix.
For example the following is true:
IsMatrix<Matrix<int, 2, 3>>::value && IsMatrix<Mat22>::value && IsMatrix<Mat33>::value
